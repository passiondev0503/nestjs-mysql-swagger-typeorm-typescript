<h1 align="center"> :small_red_triangle: Simple Nest.js CRUD</h1>



## Description

Basic CRUD with requests authentication made with Nest.js.

## :gear: Technologies
* TypeORM
* Swagger
* MySQL
* TypeScript

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
